# ⌨️ Simple Keylogger (Educational Purpose)

## 📌 Description
This project demonstrates basic keylogging using Python for learning purposes.

## ⚠️ Ethical Consideration
This tool is strictly for **educational use only**.
Do not use on any system without permission.

## 🛠 Tools Used
- Python
- pynput

## 🚀 Features
- Records keystrokes
- Saves output to a file
- Stops logging using ESC key

## ▶️ How to Run
pip install pynput
python keylogger.py


## 📚 Learning
- Keyboard event handling
- File logging
- Security awareness
